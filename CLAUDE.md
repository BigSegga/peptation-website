# Peptation Website

> Marketing website templates and assets for PEPTATION brand.

## Overview
Static website templates and assets for the PEPTATION marketing site. Contains HTML templates, CSS, and product images.

## Structure
```
assets/
├── css/
│   ├── main.css
│   └── variables.css
└── images/
    ├── logo-peptation.png
    └── product-*.jpg

templates/
├── pages/
│   ├── index.html
│   ├── product.html
│   ├── how-it-works.html
│   ├── shop.html
│   └── contact.html
└── woo-templates/
    ├── cart.html
    └── checkout.html
```

## Key Files
- `ARCHITECTURE.md` — Technical documentation
- `README.md` — Project overview
- `templates/pages/` — Main page templates
- `assets/css/variables.css` — Brand colors and styles
